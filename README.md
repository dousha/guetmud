# 桂电 MUD

## 简介

### 什么是 MUD

可以简化地理解为一种**多人在线角色扮演**游戏，和当前的 MMORPG 的概念一致。

其与 MMORPG 的不同在于，MUD 是一个更轻量级、更灵活的实现。

### 和桂电有什么关系

地图在桂电，团队在桂电，以上。

### 游戏形式

开局什么都没有，要致富，先读书！
